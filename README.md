## Тематика сайта, название

Веб-сервис для знакомств с верификацией личности и системой совместимости (сайт знакомств).

Название проекта: PickMe (от англ. "pick me" — выбери меня)

Суть: Платформа для  знакомств, где пользователи проходят верификацию, заполняют детальную анкету предпочтений и находят совместимых партнеров на основе алгоритма match.

## Цель и задачи системы

Цель: Разработать безопасный веб-сервис знакомств, который минимизирует риск мошенничества и фейковых аккаунтов, предоставляя пользователям качественный подбор партнеров на основе их предпочтений.

Задачи системы:

1. Обеспечить регистрацию и аутентификацию пользователей с верификацией личности (паспорт + реальное фото).

2. Реализовать создание и редактирование анкеты (внешность, интересы, цели знакомства) с возможностью выбора предпочтений по партнеру (цвет волос/глаз, возраст и т.д.) или опции "без разницы".

3. Разработать систему поиска и match'а (взаимные лайки) на основе совместимости пользователей.

4. Создать чаты для общения между совпавшими пользователями.

5. Реализовать анонимные признания с автоматическим раскрытием личности при взаимной симпатии.

## Содержание сайта

Страница 1. Главная — описание сервиса, преимущества, призыв к регистрации.

Страница 2. Вход / Регистрация — формы с валидацией, восстановление пароля.

Страница 3. Анкета (заполнение/редактирование) — многостраничная форма: основное, внешность, интересы, предпочтения партнера.

Страница 4. Моя анкета (профиль) — просмотр своей анкеты, кнопка редактирования.

Страница 5. Лента поиска — карточки пользователей (фото, имя, возраст, % совместимости), фильтры, кнопки лайк/дизлайк.

Страница 6. Match'и — список пользователей, с которыми есть взаимная симпатия.

Страница 7. Чат — окно переписки с конкретным пользователем.

Страница 8. Список чатов — все активные диалоги.

Страница 9. Анонимные признания — входящие признания, форма отправки нового.

Страница 10. Новости о парах — лента новостей со знаменитыми парами.

Страница 11. Друзья — список друзей, запросы, статусы.

Страница 12. Уведомления — центр уведомлений.

Страница 13. Настройки — смена пароля, тема оформления, приватность.

Страница 14. Админ-панель — модерация анкет, верификация, управление пользователями.


## Функциональные возможности

* Аутентификация и безопасность
Регистрация и вход по email.
Верификация личности.

* Анкета (CRUD)
Создание и редактирование анкеты: основное, внешность, интересы, цели.
Предпочтения партнера с опцией "без разницы".
Статус отношений (активный поиск / занят / есть пара / женат).

* Поиск и Match
Лента поиска с карточками и процентом совместимости.
Фильтрация по возрасту, городу, интересам.
Лайки/дизлайки с уведомлением о match'е.

* Чаты
Приватные чаты между совпавшими пользователями.

* Анонимные признания
Отправка признания без раскрытия личности.

Раскрытие личности при взаимном лайке.

* Друзья и уведомления
Запросы в друзья, отображение статуса друга.
Уведомления: новый match, признание, изменение статуса друга.
